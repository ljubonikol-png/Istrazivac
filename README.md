\# Istrazivac



\## Opis

Istrazivac je web crawler i arhiver za lokalne ili online potrebe.  

Omogućava:

\- Crawl web stranica sa WARC zapisom

\- Payload, CSV i JSON export

\- Pretragu i filtriranje fajlova

\- Login / Free i Premium korisnike

\- Streamlit web interfejs



\## Pokretanje

1\. Deploy na Streamlit Cloud:

&nbsp;  - Link: https://share.streamlit.io

&nbsp;  - Poveži ovaj repo

&nbsp;  - Main file path: `research.py`



2\. Lokalno pokretanje:

```bash

python research.py web
